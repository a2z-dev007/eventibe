# Eventibe

A premium event and venue booking platform built with modern web technologies, offering high-end user experiences, seamless animations, and robust functionality.

## 🚀 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **UI Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Components:** [Radix UI Primitives](https://www.radix-ui.com/)
- **State Management & Data Fetching:** [TanStack Query](https://tanstack.com/query/latest) & [Axios](https://axios-http.com/)
- **Animations & Interactions:** [Motion](https://motion.dev/), [Lenis](https://github.com/darkroomengineering/lenis) (Smooth Scrolling), Embla Carousel
- **Forms & Data:** `react-hook-form`, `react-datepicker`, `react-select`
- **AI Integration:** `@google/genai`

## ✨ Key Features

- **Venue & Event Discovery:** Comprehensive search functionality with shimmer loading states and detailed venue result cards.
- **Premium UI/UX:** High-end product showcases with horizontal scrolling, pinned showcases, and staggered feature grids.
- **Interactive Elements:** Smooth scrolling across the application, dynamic "How it Works" sections, and premium date pickers.
- **360° Virtual Tours:** Immersive room and venue explorations.
- **User Wishlists:** Functional wishlist system for users to save favorite venues and packages.
- **Seamless Booking:** Responsive and polished event quote modals for easy inquiries.

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (version 20+ recommended) installed.

### Installation

1. Copy the repository to your local machine:
   ```bash
   git clone <repository-url>
   cd eventibe
   ```

2. Install the dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Set up environment variables:
   Copy the example environment file and fill in the necessary keys.
   ```bash
   cp .env.example .env.local
   ```

### Running the Development Server

Start the development server with:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## 📁 Available Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the app for production.
- `npm run start` - Runs the built app in production mode.
- `npm run lint` - Lints the codebase using ESLint.

Eventibe is designed with an emphasis on **Rich Aesthetics**. It heavily utilizes tailored modern typography, curated seamless color palettes, and subtle micro-animations that make the interface feel responsive and alive, providing users with a truly premium experience.
